# Stop W.A.C.H. App

Beende alle laufenden W.A.C.H. Prozesse und Backends.

## Verwendung

```
/stop-app
```

## Aktion

1. Beende laufende Flutter-Prozesse
2. Beende Backend-Services (falls vorhanden)

## Befehle

```bash
# Flutter Hot-Reload beenden: 'q' in Terminal eingeben
# Oder Prozess killen falls nötig

# Zukünftig für Backends:
# docker compose down
```
